# Musterloesung A1: Kursstart KI/ML und Python-Einstieg

## Kurzloesung Begriffe

- KI: Oberbegriff fuer Systeme, die intelligente Aufgaben ausfuehren.
- Machine Learning: Teilgebiet der KI, das Muster aus Daten lernt.
- Deep Learning: Teilgebiet des ML mit neuronalen Netzen und mehreren Schichten.

## Beispiel-Reflexion (Auszug)

Ein Notebook fuehrt Code schrittweise aus. In der geaenderten Zelle wurden Listenwerte angepasst und der neue Durchschnitt berechnet. Das ist datenbasiert, weil das Ergebnis von den eingegebenen Daten abhaengt. Ein fester Wenn-Dann-Block waere regelbasiert.
