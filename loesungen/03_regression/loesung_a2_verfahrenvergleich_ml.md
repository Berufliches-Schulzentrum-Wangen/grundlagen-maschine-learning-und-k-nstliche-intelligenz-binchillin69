# Musterloesung A2: Verfahrensvergleich ML

## Kernzuordnung

- Preisprognose Wohnungen -> Regression
- Kundensegmentierung -> Clustering (k-Means)
- Kreditentscheidung ja/nein -> Klassifikation (Entscheidungsbaum oder k-NN)
- Monatsumsatzprognose -> Regression

## Beispielbegruendungen

- Regression eignet sich bei kontinuierlichen Zielwerten wie Euro-Betraegen.
- k-Means eignet sich fuer unbeschriftete Daten, wenn Gruppenstrukturen gesucht werden.
- Entscheidungsbaeume sind gut erklaerbar, koennen aber bei Ueberanpassung instabil werden.
- k-NN ist einfach, aber sensibel gegen Skalierung und Wahl von k.

## Bewertungshinweise

- Alle vier Verfahren muessen korrekt differenziert werden.
- Begruendungen muessen Zielvariable und Datencharakter beruecksichtigen.
