# Musterloesung A4: Neuronale Netze und KI-Entwicklungen

## Erklaerung neuronales Netz (Kurzfassung)

Ein neuronales Netz besteht aus Eingabeschicht, verdeckten Schichten und Ausgabeschicht. Eingaben werden gewichtet weitergegeben. Ueber Aktivierungsfunktionen entstehen Zwischenergebnisse. Beim Training wird der Fehler bestimmt und ueber Backpropagation zur Anpassung der Gewichte genutzt.

## Beispielhafte Entwicklungen (Einordnung)

- Multimodale Assistenten -> Deep Learning/Language Processing
- Lokale On-Device-Modelle -> Deep Learning
- KI-gestuetzte Prozessautomatisierung im Vertrieb -> ML/Language Processing

## Chancen-Risiken (Beispiele)

- Chance: Effizienz, bessere Prognosen
- Risiko: Bias, Datenschutz, mangelnde Erklaerbarkeit
